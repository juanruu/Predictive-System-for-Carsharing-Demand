# Predictive System for Carsharing Demand

Hourly carsharing demand prediction framework with two complementary approaches:

1. **XGBoost** -- Baseline model for Milan using ISTAT Census (ACE) zones, temporal features, and autoregressive lags. Trained with Tweedie loss.
2. **LSTM** -- Multi-city recurrent model trained on 9 European cities with H3 hexagonal spatial discretisation. Supports zero-shot transfer and fine-tuning to unseen cities (evaluated on Munich).

Based on the [car2go dataset](https://doi.org/10.1140/epjds/s13688-019-0186-8) (Boldrini et al., 2019): ~803K cleaned trips across 10 European cities.

## Repository structure

```
datasets/               Raw input data (not tracked -- see below)
codes/                  Notebooks and utility modules
  trip_cleaning.py        Cleaning filters (distance, duration, geo outliers, duplicates)
  01_data_cleaning.ipynb  Load raw trip files, apply cleaning, save Parquet
  02_xgboost_pipeline.ipynb  Full XGBoost pipeline (spatial join, features, training, SHAP)
  03_lstm_dataset.ipynb   H3 assignment, demand aggregation, temporal features
  04_lstm_model_training.ipynb  LSTM architecture definition and pretraining
  05_lstm_evaluation.ipynb      Evaluation, per-city metrics, fine-tuning on Munich
results/                Intermediate datasets and model outputs
  trips_cleaned.parquet   Cleaned trip dataset (generated by notebook 01)
  xgboost/                XGBoost dataset, predictions
  lstm/                   H3 dataset, model checkpoints, predictions
figures/                Publication-quality figures referenced in the thesis
```

## Data download

The following files are required but not tracked in Git due to their size. Place them in the indicated paths:

### Trip data (~311 MB)

Download `cs_datasets.zip` from the [Releases](https://github.com/juanruu/Predictive-System-for-Carsharing-Demand/releases) page of this repository, unzip it, and place the 10 city `.txt` files in:

```
datasets/cs_datasets/
  amsterdam.txt
  berlin.txt
  firenze.txt
  kobenhavn.txt
  milano.txt
  muenchen.txt
  roma.txt
  stockholm.txt
  torino.txt
  wien.txt
```

### ISTAT Census 2011 -- Lombardy (~84 MB shapefile + ~5 MB CSV)

Download from [ISTAT](https://www.istat.it/it/archivio/104317):

```
datasets/lombardia_shapefile/
  R03_11_WGS84.shp
  R03_11_WGS84.shx
  R03_11_WGS84.dbf
  R03_11_WGS84.prj

datasets/R03_indicatori_2011_areecensimento.csv
datasets/tracciato_2011_areecensimento.csv
```

## Execution order

Run the notebooks sequentially:

1. `01_data_cleaning.ipynb` -- produces `results/trips_cleaned.parquet`
2. `02_xgboost_pipeline.ipynb` -- produces XGBoost dataset, model, predictions, and figures
3. `03_lstm_dataset.ipynb` -- produces `results/lstm/dataset_h3_multicidad.parquet`
4. `04_lstm_model_training.ipynb` -- trains LSTM, saves checkpoint
5. `05_lstm_evaluation.ipynb` -- evaluates zero-shot and fine-tuned transfer

## Key results

| Model | MAE | RMSE | R2 |
|-------|-----|------|----|
| XGBoost (Milan) | 0.9247 | 1.4663 | 0.5660 |
| LSTM (Milan val) H3 -> ACE | 1.99 | 3.16 | 0.7659 |
| LSTM zero-shot (Munich) | 0.9845 | 1.6281 | 0.6375 |
| LSTM fine-tuned (Munich, 7 days) | 0.9590 | 1.5712 | 0.6632 |
